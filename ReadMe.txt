After using any of the commands below,
pressing on the generated image and pressing 's' will save the new image.

# rotate an image
python3 edit_image.py -p "dog.jpg" -option rotate

# draw a red line on an image
python3 edit_image.py -p "dog.jpg" -option line

# black-and-white image
python3 edit_image.py -p "dog.jpg" -option bw

# apply green mask
python3 edit_image.py -p "dog.jpg" -option mask
